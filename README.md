## Requirements
- PHP 8.3 or newer
- WSL (For Windows)
- Composer 2.7.1 or later 

## Installation

### For Linux, BSD etc
```bash
mkdir /path/to/your/game
composer init
...
composer require atatusoft-ltd/sendama-engine
```

### For Windows
From the WSL terminal follow Linux instructions

### OSX
```bash
mkdir /path/to/your/game
composer init
...
composer require atatusoft-ltd/sendama-engine
```
